Open an F\+P\+GA object

Acquires ownership of the F\+P\+GA resource referred to by \textquotesingle{}token\textquotesingle{}.

Most often this will be used to open an accelerator object to directly interact with an accelerator function, or to open an F\+P\+GA object to perform management functions.

Relinquishes ownership of a previously \mbox{\hyperlink{access_8h_addde6b2bafcd6632a2c0b595c6bc0ef3}{fpga\+Open()}}ed resource. This enables others to acquire ownership if the resource was opened exclusively. Also deallocates / unmaps M\+M\+IO and U\+Msg memory areas.
